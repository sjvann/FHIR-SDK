## 🔧 疑難排解

### 常見問題

#### 序列化/反序列化錯誤
- **原因**：來源 JSON/XML 格式錯誤，或 `resourceType` 不匹配。
- **解決方案**：使用線上工具檢查格式，並確認反序列化的類型正確。

#### 驗證失敗
- **原因**：資源的屬性不符合當前宣告的 FHIR 版本規範。
- **解決方案**：檢查 `ValidationResult.Issues` 提供的詳細錯誤資訊，並對照官方文件進行修正。
